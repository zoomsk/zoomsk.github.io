# 常用

>单行文本省略号...

``` css
overflow: hidden;
text-overflow:ellipsis;
white-space: nowrap;
```

>多行文本省略号...

``` css
overflow : hidden;
text-overflow: ellipsis;
display: -webkit-box;
-webkit-line-clamp: 2;
-webkit-box-orient: vertical;
```



>常用a标签功能

     <!-- 开启电话功能： -->
    <a href="tel:123456">123456</a>

    <!-- 开启短信功能： -->
    <a href="sms:123456">123456</a>
